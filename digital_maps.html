Great Smoky Mountains Trout Streams

The map shows the distribution of various species of trout in the Great Smoky Mountains National Park. Streams are identifiable when hovering over the stream with the mouse pointer. The popup info box displays stream name, fish type(s) present, the elevation of the stream, the watershed, park district, and state. Nearby trail names and parking areas become visible upon zooming in on portions of the map.

Data source: National Park Service - Great Smoky Mountains
The map was created using html, javascript, and Leaflet.
Github repository: github.com/MarkCruse/smoky-mountains-trout-streams


Probability of a White Christmas

A heatmap utilizing historical climate data to show what areas of the U.S. are most likely to have a white Christmas.

Data source: White Christmas? An Application of NOAA’s 1981–2010 Daily Normals
The map was created using html, javascript, and Mapbox GL JS.
Github repository: github.com/MarkCruse/white-christmas




Heritage Trees of Portland

Heritage Trees in Portland, Oregon are trees that have been formally recognized by City Council for their unique size, age, historical or horticultural significance.

The map was created using a collaborative development workflow. This means that multiple people were able to work on the project, and their changes were automatically merged into the master branch using Github.

Data source: Esri Portland R&D Center Open Data
The map was created using html, javascript, and the Leaflet library.
Github repository: github.com/MarkCruse/heritage-trees-portland




Tracking Turtles

The migratory movements of endangered sea turtles being tracked by satellite provide insight to marine biologists and conservationists. This map is intended to showcase the many geo-referenced datasets that are publicly accessible. The 20 years worth of data encompasses over 19,000 telemetry points, acquired by 13 organizations and/or individuals, showcasing 7 species of sea turtles.

The map was created using html, javascript, and D3 Javascript library.
Data source: State of the World's Sea Turtles
Github repository: github.com/MarkCruse/turtletracks




2016 Adult Uninsured Rates

Since the Affordable Care Act took effect, millions of Americans have been able to sign up for quality, affordable health insurance. The data represented in this map shows the 2016 non-elderly adult uninsured rates in each county across the United States.

Data source: Enroll America.
The map was created using html, Javascript, and D3 Javascript library.
Github repository: github.com/MarkCruse/adults_uninsured/




U.S. Power Plant Energy Output

The map shows the relative magnitudes of power plants scaled by their total electric capacity output, with filters available to view data by fuel consumption. Clicking on a point within the map will reveal the data for the plant in a pop-up window.

The map was created using html, javascript, and the Leaflet javascript library.
Data source: U.S. Energy Information Administration.
Github repository: github.com/MarkCruse/map672_power_plants/
